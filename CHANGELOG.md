# deskpals Changelog

## [Unreleased]

### Added

- **Change Sprite** command: searchable list of all Pokemon (Gen 1–4) with shiny toggle dropdown
- **Change Sprite** command: sprite preview icons displayed next to each Pokemon in the list
- **Change Sprite** command: custom sprite discovery from `~/Library/Application Support/deskpals/CustomSprites/`
- **Toggle Visibility** command: one-click show/hide for deskpals overlay
- **Resize Area** command: form to set overlay dimensions (200–2000px)
